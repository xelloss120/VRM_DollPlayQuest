■これなに？

OculusQuest2でVRMを使ってお人形遊びができます。
もしかしたらOculusQuestでも動くかもしれません。（動確はしてません。）

PCVR版とPC版があります。
PCVR版：https://120byte.booth.pm/items/1099618
PC版：https://120byte.booth.pm/items/1654585
